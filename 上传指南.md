# GitHub仓库上传指南

## 🚀 快速上传步骤

### 步骤1：创建GitHub仓库
1. 登录GitHub
2. 点击右上角"+" → "New repository"
3. 填写信息：
   - **Repository name**: `qwen-whitebox-experiment`
   - **Description**: "White-boxing memory modules of Qwen2.5-0.5B-Instruct: 80x retrieval acceleration, fully editable knowledge base"
   - **Public** ✅ (公开)
   - **Initialize with README** ❌ (不勾选，我们有自己的README)
   - **Add .gitignore** ❌ (不勾选，我们有自己的.gitignore)
   - **Choose a license** ❌ (不勾选，我们有LICENSE文件)
4. 点击"Create repository"

### 步骤2：上传文件（方法A：命令行）
```bash
# 1. 进入目录
cd /home/baiji/实验备份/github_repo

# 2. 初始化Git
git init
git add .
git commit -m "Initial commit: Qwen white-box experiment v1.0.0

- 80× retrieval acceleration (FAISS vs PyTorch)
- 275% discriminative power improvement
- Fully editable knowledge base
- C adapter implementation (1.21ms/query)
- Complete end-to-end Q&A system"

# 3. 连接到GitHub
git remote add origin https://github.com/你的用户名/qwen-whitebox-experiment.git

# 4. 推送到GitHub
git branch -M main
git push -u origin main
```

### 步骤2：上传文件（方法B：网页上传）
1. 在GitHub仓库页面，点击"Upload files"
2. 拖拽`github_repo/`目录所有文件到上传区域
3. 填写提交信息（同上）
4. 点击"Commit changes"

## 🔧 故障排除

### 常见问题

#### 问题1：文件太大上传失败
```bash
# 检查文件大小
find . -type f -size +50M

# 如果发现大文件，从Git中移除
git rm --cached 大文件路径
# 添加到.gitignore
echo "大文件路径" >> .gitignore
```

#### 问题2：Git LFS警告
```bash
# 如果GitHub提示使用Git LFS，但我们的文件很小
# 可以忽略，或安装Git LFS
git lfs install
git lfs track "*.pt"
git lfs track "*.bin"
git add .gitattributes
```

#### 问题3：权限问题
```bash
# 检查远程地址
git remote -v

# 更新远程地址
git remote set-url origin https://github.com/你的用户名/qwen-whitebox-experiment.git

# 使用SSH（如果配置了）
git remote set-url origin git@github.com:你的用户名/qwen-whitebox-experiment.git
```
